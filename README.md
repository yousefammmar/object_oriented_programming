## Topics Covered
 
### 01 · C++ Basics
Functions, loops, arrays, pointers, references, dynamic memory (`new`/`delete`), strings.
 
**Key concepts:** Variables & data types · For/while/do-while · 1D & 2D arrays · Pointers & references · Dynamic memory
 
### 02 · Class Basics
Class definition, access modifiers, setters/getters, constructors, destructors, copy constructor, `.h`/`.cpp` file separation, arrays of objects.
 
**Key concepts:** `public`/`private` access · Default & parameterised constructors · Initialiser list syntax · Copy constructor · Destructor · Header-source separation
 
### 03 · Const & Static
Constant member variables, constant member functions, static member variables, static member functions.
 
**Key concepts:** `const` after parameter list · Must init `const` in initialiser list · `static` defined outside class · `ClassName::method()` call syntax
 
### 04 · Composition
The has-a relationship — objects as members of other objects. Member object constructor ordering.
 
**Key concepts:** Class containing objects of other classes · Member object constructor runs first · Initialiser list construction · Example: `Person` has-a `Date`
 
### 05 · Labs & Practical Work
Ten labs progressing from C++ revision through pointers, classes, constructors, const/static, composition, operator overloading, inheritance, and polymorphism.
